# CIT130-Obj-Orient-Prog-1-Java
Assignments from CIT130 - Object Oriented Programming - Java

Assignment Prompt:
Implement a face that smiles/frowns. The frown and smile must be initiated by a button.
For example if your face starts out smiling there should be a frown button that makes it frown. 
When the face is frowning the button should be changed to smile. 
(you can reset the text on the button when you change images)
